The are different programming languages that involes web development

HTML

HTML is the standard markup Language used for creating web pages.It provides the structure and content of a webpage.

CSS

CSS is used to style the HTML elements, controlling the layout, colors, fonts, and other visual aspects of a webpage.

JAVASCRIPT

Javascript is a programming Language that adds interactivity and dynamic behavior to web pages. It can be used to create animations, handle user input, and manipulate the webpage content in real-time.
